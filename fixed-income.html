Get Started

Intallation

Download anaconda and create the proper enviroment. Download the files from the repository. Finally, activate the enviroment and run the jupyter-notebook server.

Treasury yields

The file data.csv contains historical yields from 1962-01-02 until 2018-07-18. To get the most recent information download the newest data from the federal reserve system. Process the file and save as data.csv.

Jupyter-notebook

Choose one notebook and follow the steps to understand what the code is doing.

Bootstrap and Interpolation

Bootstrap historical yields and interpolate the term structure.

Vasicek

Use the Vasicek model to value interest rate options

Black-Derman-Toy

Use the Black-Derman-Toy model to value interest rate options

Learning Process

The fitting process for the BDT model may require additional parameter setting. Use this visualization to understand how the model trainning is progressing.
