Portfolio Optimization

Study different allocation strategies

Get Started

Intallation

Download anaconda and create the proper enviroment. Next, activate the enviroment and run the jupyter-notebook server.

Build Dataset

Compile individual csv files, which can be obtained on google-finance or yahoo, into one unique dataset. Individual csv files must have at least two columns named 'Date' and 'Close'. You can skip this step if you have a dataset ready, just save it as 'Daily_closing_prices.csv'

Historical Data

Plot the time series and check expected daily returns and volatility.

Efficient frontier

Visualize the efficient frontier along with where each portfolio and stocks are located with respect to volatility and expected return. Supported portfolios are:

  • Equal risk contribution
  • Maximum sharpe
  • Minimum variance
  • Maximum return
  • Equal weights
