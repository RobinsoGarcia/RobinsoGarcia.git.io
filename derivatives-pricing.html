Get Started

Intallation

Download anaconda and create the proper enviroment. Next, activate the enviroment and then pip install the module using the git's repository address.

Configure quandl API key

This script leverages quandl to automatically pull stock history form the web. Quandl requires an API key that can be obtained after signing-up at quandl. Follow these steps to work with your personal APIkey.

Option pricing

This command line python program runs a simulation of future stock prices usng historical data and a brownian motion model. The result of N simulations is then used to calculate the value of an option. This implementation calculates the prices for european and american call and put options. However, it can be easily extended to value exotic options.

Help

The python script on this repository supports a -h (help) positional argument. Follow these steps to udnerstand the function of each parameter that needs to be parsed.

Download and run the PricingNotebook to visualize what's under the hood.
